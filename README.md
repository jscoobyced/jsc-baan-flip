# Baan Flip Frontend Application

[![Build](https://github.com/jscoobyced/jsc-baan-flip/actions/workflows/build.yml/badge.svg)](https://github.com/jscoobyced/jsc-baan-flip/actions/workflows/build.yml)
[![codecov](https://codecov.io/gh/jscoobyced/jsc-baan-flip/graph/badge.svg?token=UYT5AT2V75)](https://codecov.io/gh/jscoobyced/jsc-baan-flip)

Connecting homeowners looking to sell with experienced property flippers. Simplify the process, maximize value, and find your perfect match.

## Features

- Connecting homeowners with experienced property flippers
- Finding and hiring renovation contractors
- Searching for properties to flip with detailed listings
- Comprehensive property management tools
- User-friendly interface for home sellers and flippers
- Dynamic content management system
